# _example contributor

This folder is a reference example showing how a contributor namespace is
laid out. Use it as a model when adding your own `contributors/<your-handle>/`
folder.

```text
_example/
├── agents/
│   └── hello-agent/
│       ├── metadata.yaml
│       ├── README.md
│       └── agent.md
└── skills/
    └── hello-skill/
        ├── metadata.yaml
        ├── README.md
        └── skill.md
```

See the top-level [README](../../README.md) and the
[`templates/`](../../templates/) folder for the conventions every entry
follows.
