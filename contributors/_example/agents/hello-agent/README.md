# hello-agent

## Overview

`hello-agent` is a minimal, tool-agnostic example that demonstrates the
folder structure, metadata, and documentation every shared agent uses. It
greets the user and points them at the repository conventions.

## Target tool

Tool-agnostic. The prompt in [`agent.md`](./agent.md) can be pasted into any
agent runtime that accepts a system prompt.

## Usage

1. Open [`agent.md`](./agent.md).
2. Copy its contents into your agent tool of choice as the system prompt.
3. Start a conversation; the agent introduces itself and the repository.

## Configuration

None.

## Examples

- "Hello" -> the agent greets you and links to the contribution guide.
