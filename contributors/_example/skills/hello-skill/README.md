# hello-skill

## Overview

`hello-skill` is a minimal, tool-agnostic example skill. It demonstrates the
folder structure and metadata every shared skill uses by summarizing the
repository conventions on demand.

## Target tool

Tool-agnostic. The procedure in [`skill.md`](./skill.md) can be adapted to any
tool that supports skills or reusable instructions.

## Usage

1. Open [`skill.md`](./skill.md).
2. Register or paste the procedure into your tool of choice.
3. Invoke it whenever you need a refresher on repository conventions.

## Configuration

None.

## Examples

- "How do I add a new skill?" -> the skill lists the required folder, metadata,
  and documentation files.
